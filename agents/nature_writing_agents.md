# Nature Writing Agent 定义

## Agent 列表

### 1. intake_agent

**职责**：配置访谈

**能力**：
- 收集论文信息
- 确定目标期刊
- 评估创新性
- 制定写作计划

**输入**：
- 用户研究
- 目标期刊
- 写作需求

**输出**：
- 论文配置记录
- 写作计划
- 期刊要求
- 创新性评估

### 2. literature_strategist_agent

**职责**：搜索策略和文献筛选

**能力**：
- 制定搜索策略
- 执行文献搜索
- 筛选相关文献
- 评估文献质量

**输入**：
- 研究问题
- 关键词
- 数据库选择

**输出**：
- 搜索策略文档
- 文献列表
- 筛选结果
- 质量评估

### 3. structure_architect_agent

**职责**：论文结构选择

**能力**：
- 分析论文类型
- 选择论文结构
- 设计章节大纲
- 分配字数

**输入**：
- 论文类型
- 研究内容
- 期刊要求

**输出**：
- 论文结构
- 章节大纲
- 字数分配
- 写作顺序

### 4. argument_builder_agent

**职责**：声明-证据链构建

**能力**：
- 提取核心声明
- 识别支持证据
- 构建论证逻辑
- 检查论证完整性

**输入**：
- 研究发现
- 支持数据
- 文献证据

**输出**：
- 声明列表
- 证据映射
- 论证逻辑
- 完整性评估

### 5. draft_writer_agent

**职责**：分章节撰写

**能力**：
- 撰写各章节
- 应用 Nature 风格
- 整合引用
- 保持一致性

**输入**：
- 论文结构
- 声明-证据链
- 写作规范

**输出**：
- 各章节草稿
- 引用整合
- 风格检查
- 一致性验证

### 6. citation_compliance_agent

**职责**：引用验证

**能力**：
- 验证引用格式
- 检查引用完整性
- 评估引用质量
- 生成引用报告

**输入**：
- 引用列表
- 引用格式要求

**输出**：
- 引用验证报告
- 格式检查结果
- 质量评估
- 改进建议

### 7. abstract_bilingual_agent

**职责**：双语摘要

**能力**：
- 生成中文摘要
- 生成英文摘要
- 确保双语一致
- 优化摘要质量

**输入**：
- 论文内容
- 研究发现

**输出**：
- 中文摘要
- 英文摘要
- 一致性检查
- 质量评估

### 8. peer_reviewer_agent

**职责**：模拟双盲审稿

**能力**：
- 模拟审稿人
- 评估论文质量
- 提供改进建议
- 生成审稿报告

**输入**：
- 论文稿件
- 审稿标准

**输出**：
- 审稿报告
- 质量评分
- 改进建议
- 决策建议

### 9. visualization_agent

**职责**：出版级图表

**能力**：
- 设计图表
- 应用 Nature 标准
- 生成图表代码
- 优化图表质量

**输入**：
- 数据
- 图表需求
- Nature 标准

**输出**：
- 图表设计
- 图表代码
- 图表文件
- 质量评估

### 10. revision_coach_agent

**职责**：修订路线图

**能力**：
- 分析审稿意见
- 制定修订计划
- 指导修订过程
- 评估修订效果

**输入**：
- 审稿意见
- 论文稿件

**输出**：
- 修订路线图
- 修订指导
- 效果评估
- 改进建议

### 11. formatter_agent

**职责**：最终格式化

**能力**：
- 应用期刊格式
- 检查格式规范
- 生成最终版本
- 准备投稿包

**输入**：
- 论文稿件
- 期刊要求

**输出**：
- 格式化论文
- 格式检查报告
- 最终 PDF
- 投稿包

## Agent 协作

### 1. 写作流程

**Phase 0: 配置**
- intake_agent 收集信息
- 确定目标期刊
- 制定写作计划

**Phase 1: 文献**
- literature_strategist_agent 搜索
- 筛选和评估文献
- 建立文献库

**Phase 2: 结构**
- structure_architect_agent 设计
- 确定论文结构
- 分配字数

**Phase 3: 论证**
- argument_builder_agent 构建
- 提取声明
- 构建证据链

**Phase 4: 写作**
- draft_writer_agent 撰写
- 应用 Nature 风格
- 整合引用

**Phase 5: 验证**
- citation_compliance_agent 验证引用
- abstract_bilingual_agent 生成摘要
- peer_reviewer_agent 模拟审稿

**Phase 6: 定稿**
- visualization_agent 生成图表
- revision_coach_agent 指导修订
- formatter_agent 格式化

### 2. 交接规范

**交接内容**：
- 阶段输出
- 质量评估
- 问题清单
- 改进建议

**交接格式**：
```json
{
  "from_agent": "intake_agent",
  "to_agent": "literature_strategist_agent",
  "artifacts": ["paper_config.json"],
  "quality": "HIGH",
  "issues": [],
  "recommendations": []
}
```

### 3. 质量检查

**检查内容**：
- 输出完整性
- 输出准确性
- 风格一致性
- 格式规范性

**检查方式**：
- 自动检查
- 人工审核
- 用户反馈
- 质量评估

## Nature 风格规范

### 1. 语言规范

**要求**：
- 每句 ≤30 词
- 英式英语
- 主动语态为主
- 避免 AI 特征词汇

**检查项**：
- [ ] 句子长度
- [ ] 拼写规范
- [ ] 语态使用
- [ ] 词汇选择

### 2. 结构规范

**要求**：
- 倒金字塔结构
- 故事弧清晰
- 逻辑流畅
- 段落主题明确

**检查项**：
- [ ] 结构合理
- [ ] 逻辑清晰
- [ ] 过渡自然
- [ ] 主题明确

### 3. 图表规范

**要求**：
- 信息层次清晰
- 自包含性
- 色盲友好
- 符合 Nature 标准

**检查项**：
- [ ] 信息层次
- [ ] 自包含性
- [ ] 颜色规范
- [ ] 格式规范

### 4. 引用规范

**要求**：
- 作者+年份格式
- DOI 完整
- 数量符合限制
- 无自引过多

**检查项**：
- [ ] 格式正确
- [ ] DOI 完整
- [ ] 数量合理
- [ ] 引用质量
