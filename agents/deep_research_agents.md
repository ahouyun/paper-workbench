# Deep Research Agent 定义

## Agent 列表

### 1. research_question_agent

**职责**：研究问题开发

**能力**：
- 引导用户澄清研究兴趣
- 应用 FINER 标准
- 形成研究问题
- 评估问题质量

**交互方式**：
- 苏格拉底式提问
- 渐进式澄清
- 用户确认门禁

**输入**：
- 用户研究兴趣
- 初步想法
- 相关背景

**输出**：
- 研究问题
- FINER 评估
- 问题质量评分

### 2. research_architect_agent

**职责**：方法论蓝图设计

**能力**：
- 设计研究范式
- 选择研究方法
- 规划数据策略
- 评估可行性

**输入**：
- 研究问题
- 研究范围
- 资源限制

**输出**：
- 方法论蓝图
- 研究设计
- 数据收集计划
- 可行性评估

### 3. bibliography_agent

**职责**：系统文献搜索

**能力**：
- 制定搜索策略
- 执行系统搜索
- 筛选和去重
- 记录搜索过程

**输入**：
- 研究问题
- 关键词
- 数据库选择

**输出**：
- 搜索策略文档
- 文献列表
- 筛选结果
- 搜索日志

### 4. source_verification_agent

**职责**：来源验证和等级评估

**能力**：
- 验证来源真实性
- 评估证据等级
- 检查偏倚风险
- 记录质量评估

**输入**：
- 文献列表
- 验证标准

**输出**：
- 质量评估报告
- 证据等级表
- 偏倚风险评估

### 5. synthesis_agent

**职责**：跨源综合分析

**能力**：
- 整合跨源证据
- 识别主题和模式
- 分析矛盾和差异
- 识别研究空白

**输入**：
- 验证后的文献
- 质量评估

**输出**：
- 综合分析报告
- 主题矩阵
- 研究空白清单
- 综合结论

### 6. report_compiler_agent

**职责**：APA 7.0 报告编写

**能力**：
- 组织报告结构
- 撰写各部分内容
- 整合引用
- 格式化参考文献

**输入**：
- 综合分析
- 文献列表
- 写作规范

**输出**：
- 完整研究报告
- 参考文献列表
- 补充材料

### 7. editor_in_chief_agent

**职责**：Q1 期刊级编辑审稿

**能力**：
- 整体质量评估
- 编辑决策
- 修订指导
- 期刊标准对标

**输入**：
- 研究报告
- 期刊标准

**输出**：
- 编辑审稿报告
- 质量评分
- 修订建议
- 决策建议

### 8. devils_advocate_agent

**职责**：假设检测和偏见检查

**能力**：
- 挑战假设
- 检测偏见
- 压力测试论点
- 识别弱点

**输入**：
- 研究报告
- 核心论点

**输出**：
- 挑战问题
- 识别的弱点
- 改进建议

### 9. ethics_review_agent

**职责**：AI 披露和伦理审查

**能力**：
- 伦理合规检查
- AI 披露审查
- 利益冲突声明
- 数据伦理评估

**输入**：
- 研究报告
- 研究过程

**输出**：
- 伦理审查报告
- 合规状态
- 改进建议

### 10. socratic_mentor_agent

**职责**：苏格拉底式引导

**能力**：
- 引导研究思考
- 开发研究问题
- 培养批判性思维
- 提供反馈和建议

**输入**：
- 用户研究兴趣
- 对话历史

**输出**：
- 引导问题
- 反馈和建议
- 研究问题

### 11. risk_of_bias_agent

**职责**：偏倚风险评估

**能力**：
- 应用 RoB 2 工具
- 应用 ROBINS-I 工具
- 评估偏倚风险
- 生成评估报告

**输入**：
- 研究设计
- 研究数据

**输出**：
- 偏倚风险评估
- 风险等级
- 改进建议

### 12. meta_analysis_agent

**职责**：元分析和效应量

**能力**：
- 计算效应量
- 评估异质性
- 执行元分析
- 评估证据确定性

**输入**：
- 研究数据
- 效应量数据

**输出**：
- 元分析结果
- 异质性评估
- 效应量估计
- GRADE 评估

### 13. monitoring_agent

**职责**：后续文献追踪

**能力**：
- 设置文献监控
- 追踪新发表
- 检测撤稿
- 更新研究结论

**输入**：
- 研究主题
- 关键词
- 监控频率

**输出**：
- 新文献提醒
- 撤稿通知
- 更新建议

## Agent 协作

### 1. 阶段分配

**Phase 1: 问题界定**
- research_question_agent
- socratic_mentor_agent

**Phase 2: 文献搜索**
- bibliography_agent

**Phase 3: 来源验证**
- source_verification_agent
- risk_of_bias_agent

**Phase 4: 综合分析**
- synthesis_agent
- meta_analysis_agent

**Phase 5: 报告撰写**
- report_compiler_agent

**Phase 6: 编辑审稿**
- editor_in_chief_agent
- devils_advocate_agent
- ethics_review_agent

### 2. 交接规范

**交接内容**：
- 阶段输出
- 质量评估
- 问题清单
- 改进建议

**交接格式**：
```json
{
  "from_agent": "research_question_agent",
  "to_agent": "research_architect_agent",
  "artifacts": ["research_question.json"],
  "quality": "HIGH",
  "issues": [],
  "recommendations": []
}
```

### 3. 冲突解决

**冲突类型**：
- 方法论分歧
- 质量评估分歧
- 决策分歧

**解决方式**：
- 专家讨论
- 用户决策
- 多数投票
- 妥协方案
