# 完整性门禁

## 核心原则

### 1. 不可跳过

**要求**：
- 门禁检查必须执行
- 不可自动跳过
- 不可用户豁免

### 2. 全面检查

**要求**：
- 覆盖所有关键维度
- 检查标准明确
- 检查结果可追溯

### 3. 严格标准

**要求**：
- 标准不可降低
- 问题必须解决
- 质量不可妥协

---

## 门禁类型

### 1. Stage 2.5: 预审完整性门禁

**目的**：确保论文初稿符合基本完整性要求

**检查时间**：Stage 2 完成后，Stage 3 开始前

**检查内容**：

#### 数据完整性

**检查项**：
- [ ] 所有声明有数据支持
- [ ] 数据来源可追溯
- [ ] 数据分析方法适当
- [ ] 结果报告完整

**检查方法**：
- 声明-数据对照
- 数据来源验证
- 分析方法审核
- 结果完整性检查

#### 引用完整性

**检查项**：
- [ ] 所有引用有出处
- [ ] 引用格式正确
- [ ] 引用信息完整
- [ ] 无引用伪造

**检查方法**：
- 引用-出处对照
- 格式规范检查
- 信息完整性检查
- 真实性验证

#### 伦理合规性

**检查项**：
- [ ] 伦理批准已获得
- [ ] 知情同意已获得
- [ ] 隐私保护到位
- [ ] 利益冲突已声明

**检查方法**：
- 伦理批准文件检查
- 知情同意书检查
- 隐私保护评估
- 利益冲突声明检查

#### 方法可重复性

**检查项**：
- [ ] 方法描述详细
- [ ] 实验步骤清晰
- [ ] 参数设置明确
- [ ] 代码/数据可用

**检查方法**：
- 方法描述审核
- 实验步骤验证
- 参数设置检查
- 代码/数据可用性检查

#### AI 使用合规

**检查项**：
- [ ] AI 使用已披露
- [ ] AI 贡献已说明
- [ ] AI 局限性已承认
- [ ] 人类监督已确保

**检查方法**：
- AI 使用声明检查
- 贡献说明审核
- 局限性承认检查
- 监督机制验证

**决策**：
- **PASS**：所有检查项通过，进入 Stage 3
- **FAIL**：存在检查项未通过，返回 Stage 2 修改

### 2. Stage 4.5: 最终完整性门禁

**目的**：确保修订后论文无新问题，质量无下降

**检查时间**：Stage 4' 完成后，Stage 5 开始前

**检查内容**：

#### 修订完整性

**检查项**：
- [ ] 所有审稿意见已回复
- [ ] 所有修订已实施
- [ ] 修订有证据支持
- [ ] 回复信完整

**检查方法**：
- 审稿意见-回复对照
- 修订内容检查
- 证据支持验证
- 回复信审核

#### 零回归确认

**检查项**：
- [ ] 无新问题引入
- [ ] 无原有问题恶化
- [ ] 质量无下降
- [ ] 完整性无破坏

**检查方法**：
- 新问题检测
- 原问题复查
- 质量对比
- 完整性对比

#### 质量评估

**检查项**：
- [ ] 内容质量无下降
- [ ] 格式质量无下降
- [ ] 引用质量无下降
- [ ] 图表质量无下降

**检查方法**：
- 内容质量评估
- 格式质量检查
- 引用质量验证
- 图表质量审核

#### 最终确认

**检查项**：
- [ ] 所有问题已解决
- [ ] 所有建议已采纳
- [ ] 质量达标
- [ ] 准备定稿

**检查方法**：
- 问题解决确认
- 建议采纳检查
- 质量达标验证
- 定稿准备评估

**决策**：
- **PASS**：所有检查项通过，进入 Stage 5
- **FAIL**：存在检查项未通过，返回 Stage 4' 修改

## 问题处理

### 1. 问题分类

**严重问题**：
- 数据伪造
- 引用伪造
- 抄袭
- 伦理违规

**中等问题**：
- 数据不完整
- 引用不规范
- 方法不可重复
- 格式严重错误

**轻微问题**：
- 格式小错误
- 拼写错误
- 标点错误
- 排版问题

### 2. 问题处理流程

**发现问题**：
- 记录问题
- 分类问题
- 评估影响
- 确定优先级

**分析问题**：
- 分析原因
- 评估严重性
- 确定解决方案
- 制定修复计划

**修复问题**：
- 实施修复
- 验证修复
- 记录修复
- 更新文档

**确认修复**：
- 检查修复效果
- 确认问题解决
- 记录确认结果
- 更新状态

### 3. 问题升级

**升级条件**：
- 问题严重
- 影响范围大
- 解决困难
- 时间紧迫

**升级流程**：
- 报告问题
- 评估影响
- 寻求支持
- 协调解决

## 门禁豁免

### 1. 豁免原则

**原则**：
- 门禁不可豁免
- 标准不可降低
- 质量不可妥协

**例外**：
- 紧急情况
- 特殊情况
- 专家建议
- 伦理批准

### 2. 豁免流程

**申请**：
- 提交申请
- 说明理由
- 提供证据
- 专家评估

**审批**：
- 专家评审
- 风险评估
- 决策审批
- 记录存档

**执行**：
- 实施豁免
- 监控风险
- 记录过程
- 评估结果

### 3. 豁免记录

**记录内容**：
- 豁免申请
- 审批过程
- 执行情况
- 结果评估

**记录格式**：
