# 深度研究工作流

## 核心原则

### 1. 系统性

**要求**：
- 遵循预定义流程
- 每个阶段有明确目标
- 阶段间有清晰交接

### 2. 可追溯性

**要求**：
- 记录搜索策略
- 保留筛选决策
- 记录质量评估

### 3. 证据优先

**要求**：
- 基于证据的结论
- 明确证据等级
- 承认证据局限

---

## 六阶段流程

### Phase 1: 问题界定 (Scoping)

**目标**：将模糊主题转化为可研究的问题

**活动**：
1. 与用户对话澄清研究目标
2. 使用 FINER 标准评估问题
3. 确定研究范围和边界
4. 获得用户确认

**FINER 标准**：
- **F**easible (可行)：有足够资源完成
- **I**nteresting (有趣)：对领域有贡献
- **N**ovel (新颖)：填补知识空白
- **E**thical (伦理)：符合伦理规范
- **R**elevant (相关)：对理论或实践有意义

**输出**：
- 研究问题声明
- 研究范围说明
- 关键概念定义

**检查点**：用户确认后进入下一阶段

### Phase 2: 文献搜索 (Investigation)

**目标**：系统性搜索相关文献

**活动**：
1. 制定搜索策略
2. 确定数据库和来源
3. 执行搜索
4. 筛选和去重
5. 记录搜索过程

**搜索策略**：
- 关键词组合
- 布尔运算符
- 同义词扩展
- 时间范围限定

**数据库选择**：
- 学科专用：PubMed, IEEE Xplore, ACM DL
- 综合：Web of Science, Scopus, Google Scholar
- 预印本：arXiv, bioRxiv, medRxiv

**输出**：
- 搜索策略文档
- 原始文献列表
- 筛选后文献列表

### Phase 3: 来源验证 (Verification)

**目标**：评估文献质量和可靠性

**活动**：
1. 验证来源真实性
2. 评估证据等级
3. 检查偏倚风险
4. 记录质量评估

**证据等级**：
- Level 1: 系统综述/元分析
- Level 2: 随机对照试验
- Level 3: 队列研究
- Level 4: 病例对照研究
- Level 5: 病例报告/专家意见

**质量评估**：
- 研究设计
- 样本量
- 数据收集
- 分析方法
- 结果报告

**输出**：
- 质量评估报告
- 证据等级表
- 偏倚风险评估

### Phase 4: 综合分析 (Synthesis)

**目标**：整合证据，识别模式和空白

**活动**：
1. 提取关键发现
2. 识别主题和模式
3. 分析矛盾和差异
4. 识别研究空白
5. 形成综合结论

**分析方法**：
- 主题分析
- 叙述综合
- 定性比较
- 趋势分析

**输出**：
- 综合分析报告
- 主题矩阵
- 研究空白清单
- 初步结论

### Phase 5: 报告撰写 (Composition)

**目标**：撰写 APA 7.0 格式的研究报告

**活动**：
1. 组织报告结构
2. 撰写各部分内容
3. 整合引用
4. 格式化参考文献

**报告结构**：
- Title
- Abstract (150-250 words)
- Introduction
- Method
- Results
- Discussion
- References

**写作原则**：
- 清晰简洁
- 证据支持
- 逻辑连贯
- APA 7.0 格式

**输出**：
- 完整研究报告
- 参考文献列表
- 补充材料

### Phase 6: 编辑审稿 (Review)

**目标**：质量检查和修订

**活动**：
1. 编辑审稿（EIC 视角）
2. 伦理审查
3. 魔鬼代言人挑战
4. 修订和改进

**审稿维度**：
- 贡献度
- 方法论严谨性
- 证据充分性
- 写作清晰度
- 伦理合规性

**输出**：
- 审稿报告
- 修订建议
- 最终版本

**Agent**：research_question_agent

- 引导用户澄清研究目标

**职责**：

**工具**：
- 数据库 API

**门禁**：验证完成

**检查项**：
- [ ] 来源真实性确认

- [ ] 偏倚风险评估

**门禁**：综合分析完成

**检查项**：
- [ ] 主题识别完成
- [ ] 矛盾分析完成
- [ ] 研究空白识别

- [ ] 结构完整

**检查项**：

## 输出规范

### 研究报告

**格式**：APA 7.0

**长度**：3000-8000 词

**结构**：
- Title (≤12 词)
- Abstract (150-250 词)
- Introduction (背景 → 问题 → 目标)
- Method (搜索策略 → 筛选标准 → 分析方法)
- Results (主题呈现 → 证据综合)
- Discussion (解释 → 局限 → 未来方向)
- References (APA 7.0 格式)

### 文献综述

**格式**：带注释的参考文献

**长度**：1500-4000 词

**结构**：
- 引言 (研究问题和范围)
- 方法 (搜索策略和筛选)
- 主题 1 (相关研究综合)
- 主题 2 (相关研究综合)
- ...
- 研究空白 (识别和论证)
- 结论 (总结和意义)

### 快速研究简报

**格式**：简短报告

**长度**：500-1500 词

**结构**：
- 背景 (1-2 段)
- 关键发现 (3-5 个要点)
- 意义 (1 段)
- 参考文献 (精选)

## 苏格拉底引导模式

### 五层对话模型

**Layer 1: 澄清 (Clarification)**
- "你能更具体地描述你的研究兴趣吗？"
- "你希望解决什么问题？"
- "你的研究范围是什么？"

**Layer 2: 假设探查 (Assumption Probing)**
- "你做了什么假设？"
- "这些假设合理吗？"
- "如果假设不成立会怎样？"

**Layer 3: 证据/推理 (Evidence/Reasoning)**
- "你有什么证据支持这个观点？"
- "你的推理过程是什么？"
- "有没有反面证据？"

**Layer 4: 视角/观点 (Viewpoint/Perspective)**
- "从其他角度怎么看？"
- "其他研究者会怎么想？"
- "有没有替代解释？"

**Layer 5: 影响/后果 (Implication/Consequence)**
- "这个发现意味着什么？"
- "有什么实际应用？"
- "未来研究方向是什么？"

### 对话流程

1. **探索阶段**：广泛探索研究兴趣
2. **聚焦阶段**：逐步聚焦到具体问题
3. **深化阶段**：深入探讨问题的各个方面
4. **收敛阶段**：形成明确的研究问题
5. **确认阶段**：用户确认最终问题

### 退出条件

- 研究问题满足 FINER 标准
- 用户明确表示满意
- 达到最大对话轮次（60 轮）
