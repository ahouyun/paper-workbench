# 系统综述协议 (PRISMA 2020)

## 核心原则

### 1. 透明性

**要求**：
- 预注册协议
- 详细记录方法
- 报告所有决策

### 2. 可重复性

**要求**：
- 详细的方法描述
- 明确的纳入/排除标准
- 标准化的筛选流程

### 3. 全面性

**要求**：
- 系统性搜索
- 多数据库覆盖
- 灰色文献纳入

## PICOS 框架

### P - Population (人群)

**定义**：研究的目标人群

**要素**：
- 年龄
- 性别
- 健康状况
- 地理位置
- 其他特征

**示例**：
```
成人（≥18岁）2型糖尿病患者
```

### I - Intervention (干预)

**定义**：研究的干预措施

**要素**：
- 干预类型
- 剂量/强度
- 持续时间
- 实施方式

**示例**：
```
二甲双胍单药治疗
```

### C - Comparison (对照)

**定义**：比较的干预或对照

**要素**：
- 安慰剂
- 常规治疗
- 其他干预
- 无对照

**示例**：
```
安慰剂或常规治疗
```

### O - Outcome (结局)

**定义**：测量的结局指标

**要素**：
- 主要结局
- 次要结局
- 测量方法
- 测量时间点

**示例**：
```
主要结局：HbA1c 变化
次要结局：体重变化、低血糖事件
```

### S - Study Design (研究设计)

**定义**：纳入的研究设计类型

**要素**：
- 随机对照试验
- 队列研究
- 病例对照研究
- 其他设计

**示例**：
```
随机对照试验
```

## 搜索策略

### 数据库选择

**核心数据库**：
- PubMed/MEDLINE
- Embase
- Cochrane Library
- Web of Science

**学科专用数据库**：
- PsycINFO (心理学)
- CINAHL (护理)
- ERIC (教育)
- IEEE Xplore (工程)

**灰色文献**：
- ClinicalTrials.gov
- WHO ICTRP
- 研究生论文数据库
- 会议论文集

### 搜索词构建

**步骤**：
1. 识别关键概念
2. 确定同义词
3. 使用布尔运算符
4. 应用限制条件

**示例**：
```
("diabetes mellitus, type 2"[MeSH] OR "type 2 diabetes"[tiab] OR "T2DM"[tiab])
AND
("metformin"[MeSH] OR "metformin"[tiab] OR "glucophage"[tiab])
AND
("randomized controlled trial"[pt] OR "controlled clinical trial"[pt] OR "random*"[tiab])
```

### 搜索验证

**方法**：
- 已知相关研究检查
- 敏感性分析
- 专家咨询

**记录**：
- 每个数据库的搜索策略
- 搜索日期
- 结果数量
- 去重后数量

## 研究筛选

### 纳入标准

**人群**：
- 年龄范围
- 健康状况
- 地理位置
- 其他特征

**干预**：
- 干预类型
- 剂量/强度
- 持续时间
- 实施方式

**对照**：
- 对照类型
- 对照条件

**结局**：
- 主要结局
- 次要结局
- 测量方法

**研究设计**：
- 设计类型
- 发表状态
- 语言限制

### 排除标准

**常见排除**：
- 非目标人群
- 非目标干预
- 非目标结局
- 非目标设计
- 重复发表
- 无法获取全文

### 筛选流程

**Level 1: 标题摘要筛选**
- 快速筛选
- 宽松标准
- 保留所有可能相关

**Level 2: 全文筛选**
- 详细评估
- 应用纳入/排除标准
- 记录排除原因

**质量控制**：
- 双人独立筛选
- 一致性检查
- 分歧解决机制

**领域**：
1. 随机化过程
2. 偏离干预
3. 结局数据缺失
4. 结局测量
5. 选择性报告

**评估**：
- 低风险
- 有些担忧
- 高风险

### ROBINS-I (非随机研究)

**领域**：
1. 混杂偏倚
2. 选择偏倚
3. 干预分类偏倚
4. 偏离干预偏倚
5. 缺失数据偏倚
6. 结局测量偏倚
7. 选择性报告偏倚

**评估**：
- 低风险
- 中等风险
- 高风险
- 关键风险
- 无信息

### 评估流程

**双人评估**：
- 独立评估
- 一致性检查
- 分歧解决

**结果呈现**：
- 交通灯图
- 汇总图

## 数据合成

### 定性合成

**方法**：
- 叙述综合
- 主题分析
- 投票计数

**适用情况**：
- 研究异质性大
- 数据不足
- 无法进行定量合成

### 定量合成 (元分析)

**效应量测量**：
- 连续变量：均值差 (MD)、标准化均值差 (SMD)
- 二分类变量：风险比 (RR)、比值比 (OR)
- 相关性：相关系数 (r)

**合并方法**：
- 固定效应模型
- 随机效应模型
- 贝叶斯方法

**异质性评估**：
- Q 检验
- I² 统计量
- τ² 估计

**亚组分析**：
- 预定义亚组
- 探索性亚组
- 调整混杂

**敏感性分析**：
- 留一法
- 影响力分析
- 质量效应

## 证据确定性

### GRADE 框架

**降级因素**：
1. 偏倚风险
2. 不一致性
3. 间接性
4. 不精确性
5. 发表偏倚

**升级因素**：
1. 效应量大
2. 剂量反应
3. 混杂因素减少偏倚

**确定性等级**：
- 高：非常有信心效果估计接近真实效果
- 中等：效果估计可能接近真实效果，但可能明显不同
- 低：效果估计可能与真实效果明显不同
- 非常低：效果估计与真实效果可能明显不同

### 评估流程

**逐结局评估**：
- 每个主要结局单独评估
- 考虑所有降级/升级因素
- 形成证据概况表

**结果呈现**：
- 证据概况表 (Summary of Findings)
- 证据确定性评级
- 效果估计和置信区间
