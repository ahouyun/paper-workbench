# 标准解析优先级

当论文任务提到学校模板、本科论文标准、国标、参考文献格式或内置默认与真实学校要求冲突时使用。

## 解析优先级

按以下顺序应用要求：

1. 学校或学院论文模板、写作指南、答辩通知和官方表格
2. 导师、研究组、任务书、开题报告或项目特定要求
3. 教育部学术诚信和本科论文检查规则
4. 学校指定的国标
5. 内置回退默认值

**永远不要让内置默认覆盖真实学校模板。** 如果要求缺失，将回退标记为 `default_not_school_confirmed`。

## 当前公共标准基线

截至 2026-05-01：

- `GB/T 7713.1-2025` 是论文/学位论文写作结构和组成的当前公共基线。除非学校明确采用，仅作为结构参考。
- `GB/T 7714-2025` 已发布，计划于 2026-07-01 实施。过渡期内许多学校可能仍要求 `GB/T 7714-2015`。`standard-profile.yaml` 必须记录学校实际要求的版本。
- 教育部论文检查和学术诚信规则是质量和诚信基线，不是页面布局模板。

## 标准配置字段

起草前更新 `thesis-workspace/.thesis-config.yaml` 或 `standard-profile.yaml`：

- 学校、学院、专业、更新日期
- 确切模板路径或 URL
- 导师或任务书要求来源
- 参考文献样式标准和版本
- 论文写作标准版本（如学校指定）
- 非学校确认的回退项
- 仍需 Word/PDF 视觉审查的布局项

## 冲突处理

两个来源冲突时：

| 冲突 | 操作 |
|------|------|
| 学校模板 vs 内置默认 | 跟学校模板 |
| 导师要求 vs 内置章节模型 | 跟导师（学术合理时） |
| 学校要求 GB/T 7714-2015 但文档提到 2025 | 用 2015，记录学校来源 |
| 学校对某项沉默 | 用回退默认，标记为可替换 |
| 来源不明确或非官方 | 不强制执行；列为 `needs_confirmation` |

## 输出合同

解析标准时返回：

1. 确认的硬规则
2. 回退默认值
3. 冲突和选择的来源
4. 缺失的确认项
5. 需要 Word/PDF 视觉审查的布局项

在分离硬规则和回退假设前，不写最终论文正文。
