# Nature 引用格式指南

## 核心原则

### 1. 作者+年份格式

**格式**：
```
(Smith et al., 2024)
(Smith & Jones, 2024)
(Smith, Jones & Lee, 2024)
```

**规则**：
- 1-2 位作者：全部列出
- 3 位以上：第一位作者 + et al.
- 同年多篇：加字母后缀 (2024a, 2024b)

### 2. 参考文献列表

**格式**：
```
1. Smith, J. et al. Title of the paper. Journal Name Volume, Pages (Year).
2. Jones, A. & Lee, B. Title of the paper. Journal Name Volume, Pages (Year).
```

**规则**：
- 按出现顺序编号
- 作者姓名：姓 + 名首字母
- 期刊名：斜体
- 卷号：粗体
- 页码：完整范围

### 3. DOI 必需

**要求**：
- 每条引用必须有 DOI
- 格式：`https://doi.org/xxxxx`
- 无 DOI 时：提供 URL 或 PMID

## 引用类型

### 1. 期刊文章

**格式**：
```
Smith, J. et al. Title of the article. Journal Name Volume, Pages (Year).
```

**示例**：
```
1. Smith, J. et al. A novel approach to gene editing. Nature Biotechnology 42, 123-130 (2024).
2. Jones, A. & Lee, B. Machine learning in drug discovery. Science 380, 456-462 (2023).
```

### 2. 书籍

**格式**：
```
Author, A. A. Title of Book. (Publisher, Year).
```

**示例**：
```
3. Watson, J. D. Molecular Biology of the Gene. (Pearson, 2014).
```

### 3. 书籍章节

**格式**：
```
Author, A. A. Title of chapter. in Title of Book (ed. Editor, A. A.) Pages (Publisher, Year).
```

**示例**：
```
4. Smith, J. Gene editing techniques. in CRISPR Methods (ed. Johnson, K.) 45-67 (Springer, 2023).
```

### 4. 预印本

**格式**：
```
Author, A. A. Title of the paper. Preprint at arXiv/arXiv:xxxx.xxxxx (Year).
```

**示例**：
```
5. Lee, C. et al. A new transformer architecture. Preprint at arXiv/arXiv:2301.12345 (2023).
```

### 5. 网页

**格式**：
```
Author, A. A. Title of webpage. URL (accessed date).
```

**示例**：
```
6. World Health Organization. COVID-19 dashboard. https://covid19.who.int (2024).
```

### 6. 数据集

**格式**：
```
Author, A. A. Title of dataset. Repository, DOI (Year).
```

**示例**：
```
7. Smith, J. Gene expression dataset. Figshare, https://doi.org/10.6084/m9.figshare.12345 (2024).
```

## 正文引用规则

### 1. 引用位置

**规则**：
- 引用放在支持的声明之后
- 引用放在句末标点之前
- 避免引用堆砌

**示例**：
```
✅ Gene editing has improved disease treatment (Smith et al., 2024).
❌ Gene editing (Smith et al., 2024) has improved disease treatment.
```

### 2. 多篇引用

**规则**：
- 按时间顺序排列
- 用分号分隔
- 最多 3 篇

**示例**：
```
✅ Gene editing has advanced rapidly (Smith et al., 2020; Jones et al., 2022; Lee et al., 2024).
```

### 3. 作者作为句子成分

**规则**：
- 作者姓名不加括号
- 年份加括号
- 用于强调特定研究

**示例**：
```
✅ Smith et al. (2024) demonstrated that gene editing can cure genetic diseases.
```

### 4. 同一作者多篇

**规则**：
- 加字母后缀
- 按时间顺序排列

**示例**：
```
✅ Recent studies (Smith et al., 2024a, 2024b) have shown...
```

## 参考文献列表格式

### 1. 列表结构

**格式**：
```
References

1. Author, A. A. et al. Title. Journal Volume, Pages (Year).
2. Author, B. B. & Author, C. C. Title. Journal Volume, Pages (Year).
...
```

### 2. 作者姓名

**规则**：
- 姓在前，名首字母在后
- 多作者用逗号分隔
- 最后两位作者用 & 连接

**示例**：
```
Smith, J., Jones, A. & Lee, B.
```

### 3. 期刊名称

**规则**：
- 使用标准缩写
- 斜体显示
- 每个词首字母大写

**示例**：
```
Nature Biotechnology → Nat. Biotechnol.
Science → Science
Cell → Cell
```

### 4. 卷号和页码

**规则**：
- 卷号：粗体
- 页码：完整范围
- 年份：括号内

**示例**：
```
Nature Biotechnology 42, 123-130 (2024).
```

## 特殊情况

### 1. 无 DOI 文献

**处理**：
- 提供 URL
- 或提供 PMID
- 标注访问日期

**示例**：
```
8. World Health Organization. COVID-19 dashboard. https://covid19.who.int (2024).
```

### 2. 预印本

**处理**：
- 标注预印本服务器
- 提供 arXiv ID
- 说明状态

**示例**：
```
9. Lee, C. et al. A new transformer architecture. Preprint at arXiv/arXiv:2301.12345 (2023).
```

### 3. 已接受未发表

**处理**：
- 标注 "in press"
- 提供 DOI（如有）
- 说明期刊

**示例**：
```
10. Smith, J. et al. Gene editing in clinical trials. Nat. Med. (in press).
```

### 4. 同一作者同年多篇

**处理**：
- 加字母后缀
- 按标题字母顺序

**示例**：
```
11. Smith, J. et al. Gene editing advances. Nat. Biotechnol. 42, 123-130 (2024a).
12. Smith, J. et al. Gene editing challenges. Nat. Biotechnol. 42, 456-462 (2024b).
```

## 常见错误

### 1. 格式错误

**问题**：作者姓名格式错误

**正确**：
```
✅ Smith, J. et al.
❌ J. Smith et al.
❌ Smith J. et al.
```

### 2. 引用堆砌

**问题**：一句话引用多篇文献

**解决**：
- 分散引用
- 每个声明单独引用
- 按重要性排序

### 3. DOI 缺失

**问题**：引用没有 DOI

**解决**：
- 查找 DOI
- 提供 URL
- 标注 PMID

### 4. 期刊缩写错误

**问题**：期刊名称缩写不规范

**解决**：
- 使用标准缩写
- 参考 PubMed
- 保持一致
