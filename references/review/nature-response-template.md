# Nature 审稿回复模板

## 核心原则

### 1. 结构化回复

**格式**：
- Point-by-point 回复
- 每个审稿意见独立回复
- 明确标注修改位置

**结构**：
```
Reviewer Comment: [原文引用]
Response: [回复内容]
Changes Made: [修改说明和位置]
```

### 2. 证据导向

**原则**：
- 用数据和证据回复
- 引用具体位置
- 提供补充实验（如需要）

**示例**：
```
✅ We thank the reviewer for this comment. We have added new experiments 
   (Figure 3, Table 2) that address this concern. The results show...
❌ We disagree with the reviewer. Our method is clearly superior.
```

### 3. 专业礼貌

**语气**：
- 感谢审稿人的意见
- 承认合理的批评
- 保持专业和尊重

**避免**：
- 防御性语气
- 轻视审稿意见
- 过度道歉

## 回复类型

### 1. 接受并修改

**适用**：审稿意见合理，可以修改

**模板**：
```
We thank the reviewer for this insightful comment. We have revised the 
manuscript accordingly.

Changes Made:
- [Section/Page]: [具体修改内容]
- [Line Numbers]: [修改行号]
```

**示例**：
```
Reviewer Comment: The introduction lacks context on previous work in this area.

Response: We thank the reviewer for this suggestion. We have expanded the 
introduction to include a more comprehensive discussion of prior work.

Changes Made:
- Introduction, Paragraph 2: Added discussion of Smith et al. (2020) and 
  Jones et al. (2022) findings
- Lines 45-62: New text describing the evolution of this field
```

### 2. 部分接受

**适用**：部分同意，需要解释

**模板**：
```
We thank the reviewer for this comment. We partially agree with this 
suggestion and have made the following changes.

Changes Made:
- [修改内容]

Explanation:
- [解释为什么部分同意]
```

**示例**：
```
Reviewer Comment: The sample size is too small for the conclusions drawn.

Response: We thank the reviewer for raising this important point. We agree 
that a larger sample size would strengthen the conclusions. However, given 
the constraints of our study, we have:

1. Added a power analysis (Methods, Lines 123-130)
2. Acknowledged this limitation (Discussion, Lines 234-240)
3. Provided effect sizes alongside p-values

We believe these additions adequately address the concern while maintaining 
the validity of our findings.
```

### 3. 礼貌反驳

**适用**：审稿意见有误或不合理

**模板**：
```
We thank the reviewer for this comment. After careful consideration, we 
respectfully disagree for the following reasons.

Explanation:
- [理由1]
- [理由2]

Supporting Evidence:
- [证据1]
- [证据2]
```

**示例**：
```
Reviewer Comment: The authors should use Method X instead of Method Y.

Response: We thank the reviewer for this suggestion. After careful 
consideration, we respectfully maintain our choice of Method Y for the 
following reasons:

1. Method X has been shown to have limitations in our specific context 
   (Smith et al., 2020)
2. Our validation experiments show comparable performance (Table S3)
3. Method Y provides better interpretability for our research question

We have added a discussion of this choice in the Methods section 
(Lines 89-95) to provide more transparency.
```

### 4. 补充实验

**适用**：需要新数据支持

**模板**：
```
We thank the reviewer for this suggestion. We have conducted additional 
experiments to address this concern.

New Results:
- [实验描述]
- [结果说明]

Changes Made:
- [Figure/Table]: [新增图表]
- [Section]: [新增讨论]
```

**示例**：
```
Reviewer Comment: The authors should test their method on Dataset Z.

Response: We thank the reviewer for this excellent suggestion. We have 
conducted additional experiments on Dataset Z.

New Results:
- Our method achieves 85.3% accuracy on Dataset Z (Table 3)
- This is comparable to our results on the original datasets
- The results validate the generalizability of our approach

Changes Made:
- Table 3: New comparison results on Dataset Z
- Results Section: Added paragraph discussing these findings (Lines 156-165)
- Discussion: Added generalizability discussion (Lines 267-275)
```

## 特殊情况处理

### 1. 审稿人意见矛盾

**问题**：两个审稿人意见相反

**策略**：
- 分别回复每个审稿人
- 解释如何平衡
- 说明最终决定

**示例**：
```
Response to Reviewer 1: We agree that Method A has advantages in terms of 
accuracy, and we have emphasized this in the revised manuscript.

Response to Reviewer 2: We appreciate the suggestion to use Method B. 
However, given the specific requirements of our study, we have maintained 
Method A because [reasons].

To balance these perspectives, we have added a discussion comparing both 
methods (Discussion, Lines 289-300).
```

### 2. 无法完成的实验

**问题**：审稿人要求无法实现的实验

**策略**：
- 解释限制
- 提供替代方案
- 说明不影响结论

**示例**：
```
Reviewer Comment: The authors should perform experiment X to validate their 
findings.

Response: We appreciate this suggestion. However, experiment X requires 
[specialized equipment/long-term follow-up/ethical constraints] that are 
currently beyond the scope of this study.

Alternative Approach:
- We have provided computational validation (Figure S2)
- We have cited supporting evidence from related studies (References 15-18)
- We have added this as a future direction (Discussion, Lines 301-305)

We believe our current evidence is sufficient to support our conclusions.
```

### 3. 语言和写作问题

**问题**：审稿人指出语言问题

**策略**：
- 感谢指出
- 说明已修改
- 如需要，寻求专业帮助

**示例**：
```
Reviewer Comment: The manuscript has several grammatical errors and unclear 
passages.

Response: We thank the reviewer for carefully reading the manuscript. We 
have revised the entire manuscript for clarity and grammar.

Changes Made:
- Full manuscript: Professional English editing service used
- Specific passages mentioned by reviewer: [line-by-line corrections]
- Additional clarity improvements throughout
```

## 回复结构

### 1. 封面信

**内容**：
- 感谢编辑和审稿人
- 概述主要修改
- 强调论文改进

**示例**：
```
Dear Editor and Reviewers,

We thank you for the constructive comments on our manuscript. We have 
carefully considered all suggestions and have made substantial revisions 
to address each point. The major changes include:

1. [主要修改1]
2. [主要修改2]
3. [主要修改3]

We believe these revisions have significantly improved the manuscript 
and hope it is now suitable for publication in [Journal Name].

Sincerely,
[Authors]
```

### 2. 逐点回复

**格式**：
```
RESPONSE TO REVIEWER 1

Comment 1: [原文引用]
Response: [回复]
Changes: [修改说明]

Comment 2: [原文引用]
Response: [回复]
Changes: [修改说明]

RESPONSE TO REVIEWER 2

...
```

### 3. 修改标记

**方法**：
- 使用修订模式
- 或用颜色标记
- 提供行号对照

**示例**：
```
Changes in the revised manuscript are highlighted in yellow. 
Line numbers refer to the revised manuscript unless otherwise noted.
```

## 常见错误

### 1. 防御性语气

**问题**：回复过于防御

**错误**：
```
❌ We strongly disagree with the reviewer. Our method is clearly superior.
```

**正确**：
```
✅ We thank the reviewer for this comment. After careful consideration, 
   we respectfully maintain our approach because [reasons].
```

### 2. 缺乏证据

**问题**：回复没有证据支持

**错误**：
```
❌ Our method works better than existing methods.
```

**正确**：
```
✅ Our method achieves 92.3% accuracy compared to 89.1% for the best 
   existing method (Table 2, p < 0.01).
```

### 3. 不完整回复

**问题**：没有完全回应审稿意见

**错误**：
```
❌ We have addressed this concern in the revised manuscript.
```

**正确**：
```
✅ We have addressed this concern by:
   1. Adding new experiments (Table 3)
   2. Revising the discussion (Lines 234-245)
   3. Acknowledging limitations (Lines 267-275)
```

### 4. 过度道歉

**问题**：道歉过多显得不专业

**错误**：
```
❌ We sincerely apologize for this oversight. We are deeply sorry for 
   the confusion caused. We deeply regret this error.
```

**正确**：
```
✅ We thank the reviewer for identifying this issue. We have corrected 
   the error and revised the relevant sections.
```

## 自检清单

### 内容检查

- [ ] 所有审稿意见都有回复
- [ ] 回复有证据支持
- [ ] 修改位置明确标注
- [ ] 语言专业礼貌

### 结构检查

- [ ] 封面信完整
- [ ] 逐点回复格式正确
- [ ] 修改标记清晰
- [ ] 行号对照准确

### 语气检查

- [ ] 无防御性语气
- [ ] 感谢审稿人
- [ ] 承认合理批评
- [ ] 保持专业尊重

### 完整性检查

- [ ] 所有修改都已实施
- [ ] 新增内容有说明
- [ ] 补充实验有描述
- [ ] 局限性有承认
