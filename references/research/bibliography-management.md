# 文献管理指南

## 核心原则

### 1. 系统性

**要求**：
- 统一的文献收集流程
- 标准化的分类体系
- 完整的元数据记录

### 2. 可追溯性

**要求**：
- 记录来源
- 记录搜索策略
- 记录筛选决策

### 3. 可重用性

**要求**：
- 标准化格式
- 完整的引用信息
- 便于导出和共享

## 文献筛选

### 筛选流程

**Level 1: 标题摘要筛选**
- 快速评估相关性
- 宽松标准
- 保留所有可能相关

**Level 2: 全文筛选**
- 详细评估
- 应用纳入/排除标准
- 记录排除原因

### 筛选标准

**纳入标准**：
- 研究问题相关
- 人群匹配
- 干预匹配
- 结局匹配
- 设计符合

**排除标准**：
- 非目标人群
- 非目标干预
- 非目标结局
- 非目标设计
- 重复发表
- 无法获取

### 筛选记录

**筛选日志**：
```
总记录数：[数量]
Level 1 排除：[数量]，原因：...
Level 2 排除：[数量]，原因：...
最终纳入：[数量]
```

## 文献存储

### 存储方式

**本地存储**：
- 文件夹结构
- 文件命名规范
- 备份策略

**云端存储**：
- 云盘同步
- 团队共享

### 文件命名

**命名规范**：
```
[作者]_[年份]_[标题关键词].[扩展名]
```

**示例**：
```
Smith_2024_GeneEditing_Nature.pdf
Jones_2023_MachineLearning_Science.pdf
```

### 文件夹结构

**推荐结构**：
```
Literature/
├── By_Topic/
│   ├── Topic_A/
│   ├── Topic_B/
│   └── Topic_C/
├── By_Method/
│   ├── RCT/
│   ├── Cohort/
│   └── CaseControl/
├── By_Year/
│   ├── 2024/
│   ├── 2023/
│   └── 2022/
└── Unsorted/
```

## 文献阅读

### 阅读策略

**三遍阅读法**：
1. **第一遍**：标题、摘要、结论
2. **第二遍**：引言、方法、结果
3. **第三遍**：详细阅读、批判分析

**阅读顺序**：
1. 综述文章
2. 相关研究
3. 方法学文章
4. 最新研究

### 阅读笔记

**笔记内容**：
- 研究问题
- 方法设计
- 主要发现
- 局限性
- 与研究相关性

**笔记格式**：
```
研究问题：[问题]
方法设计：[设计]
主要发现：[发现]
局限性：[局限]
相关性：[评价]
```

### 阅读追踪

**追踪指标**：
- 阅读状态
- 阅读时间
- 阅读次数
- 笔记数量

**追踪工具**：
- Zotero 标签
