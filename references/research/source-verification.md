# 来源验证指南

## 核心原则

### 1. 真实性

**要求**：
- 验证来源存在
- 确认作者身份
- 核实发表信息

### 2. 可靠性

**要求**：
- 评估期刊质量
- 检查同行评审
- 评估研究设计

### 3. 相关性

**要求**：
- 评估与研究问题相关性
- 评估证据等级
- 评估时效性

---

## 验证流程

### Level 1: 基本验证

**目标**：确认来源真实存在

**检查项**：
- [ ] DOI 有效
- [ ] 期刊真实存在
- [ ] 作者可查
- [ ] 发表年份正确

**方法**：
- DOI 解析 (doi.org)
- 期刊网站查询
- 作者主页验证
- 数据库交叉验证

### Level 2: 质量验证

**目标**：评估来源质量

**检查项**：
- [ ] 期刊影响因子
- [ ] 同行评审状态
- [ ] 研究设计质量
- [ ] 样本量合理性

**方法**：
- Journal Citation Reports
- 期刊网站确认
- 研究设计评估
- 统计方法评估

### Level 3: 相关性验证

**目标**：评估与研究问题的相关性

**检查项**：
- [ ] 研究问题相关
- [ ] 人群相关
- [ ] 干预相关
- [ ] 结局相关

**方法**：
- 全文阅读
- 内容分析
- 证据等级评估

---

## 证据等级

### Level 1: 系统综述/元分析

**定义**：对多个研究的系统性综合

**质量指标**：
- 搜索策略全面
- 纳入标准明确
- 偏倚风险评估
- 合成方法适当

**评估工具**：
- AMSTAR 2
- ROBIS

### Level 2: 随机对照试验

**定义**：随机分配干预的实验研究

**质量指标**：
- 随机化方法
- 分配隐藏
- 盲法
- 失访率

**评估工具**：
- RoB 2
- Jadad 量表

### Level 3: 队列研究

**定义**：前瞻性观察研究

**质量指标**：
- 代表性
- 随访完整性
- 混杂控制
- 结局测量

**评估工具**：
- Newcastle-Ottawa 量表
- ROBINS-I

### Level 4: 病例对照研究

**定义**：回顾性比较研究

**质量指标**：
- 病例定义
- 对照选择
- 暴露测量
- 混杂控制

**评估工具**：
- Newcastle-Ottawa 量表
- ROBINS-I

### Level 5: 病例报告/专家意见

**定义**：个案描述或专家观点

**质量指标**：
- 描述详细
- 诊断明确
- 治疗合理
- 随访完整

**评估**：
- 叙述性评估
- 专家共识

## 来源类型验证

### 1. 期刊文章

**验证步骤**：
1. 确认期刊真实存在
2. 检查 ISSN
3. 验证 DOI
4. 确认同行评审状态

**红旗信号**：
- 期刊不在主流数据库
- 缺少 ISSN
- 无同行评审
- 掠夺性期刊特征

### 2. 预印本

**验证步骤**：
1. 确认预印本服务器
2. 检查版本状态
3. 查找正式发表版本
4. 评估质量

**注意事项**：
- 未经同行评审
- 可能已更新
- 需注明状态

### 3. 书籍

**验证步骤**：
1. 确认出版商
2. 检查 ISBN
3. 验证作者资质
4. 评估内容质量

**评估标准**：
- 知名出版商
- 学术专著
- 作者专家
- 内容可靠

### 4. 灰色文献

**类型**：
- 会议论文
- 学位论文
- 技术报告
- 政策文件

**验证步骤**：
1. 确认来源机构
2. 检查作者资质
3. 评估内容质量
4. 考虑时效性

---

## 掠夺性期刊识别

### 特征

**网站特征**：
- 网站设计粗糙
- 联系信息不完整
- 编辑委员会信息模糊
- 索取高额费用

**期刊特征**：
- 影响因子虚假
- 同行评审缺失
- 发表速度过快
- 范围过广

**行为特征**：
- 主动邀请投稿
- 承诺快速发表
- 费用不透明
- 撤稿政策缺失

### 检查工具

**工具列表**：
- Beall's List (已停止更新)
- DOAJ (开放获取期刊目录)
- Cabell's Scholarly Analytics
- Journal Citation Reports

**检查步骤**：
1. 查询 DOAJ
2. 检查索引状态
3. 验证影响因子
4. 评估编辑委员会

## 数据库验证

### PubMed

**验证内容**：
- PMID 存在
- 记录完整
- MeSH 标引
- 全文链接

**检查方法**：
```
https://pubmed.ncbi.nlm.nih.gov/[PMID]/
```

### Web of Science

**验证内容**：
- WOS 入藏号
- 引用次数
- 期刊影响因子
- 研究领域

**检查方法**：
```
Web of Science 数据库搜索
```

### Scopus

**验证内容**：
- Scopus ID
- 引用次数
- 作者信息
- 期刊指标

**检查方法**：
```
Scopus 数据库搜索
```

### Google Scholar

**验证内容**：
- 引用次数
- 相关文章
- 作者主页
- 全文可用性

**检查方法**：
```
Google Scholar 搜索
```

## DOI 验证

### 验证步骤

1. **格式检查**：10.xxxx 格式
2. **解析验证**：doi.org 解析
3. **内容匹配**：标题、作者匹配
4. **状态检查**：是否有效

### 验证工具

**在线工具**：
- doi.org
- CrossRef
- DataCite

**API**：
- CrossRef API
- DataCite API

### 常见问题

**DOI 不存在**：
- 检查格式
- 验证来源
- 联系作者

**DOI 指向错误**：
- 检查内容
- 报告错误
- 寻找正确 DOI

---

## 作者验证

### 验证步骤

1. **姓名检查**：拼写、格式
2. **机构验证**：当前/历史机构
3. **资质验证**：学历、职位
4. **发表记录**：其他发表

### 验证工具

**学术平台**：
- ORCID
- Google Scholar
- ResearchGate
- 学校主页

**验证方法**：
- ORCID 搜索
- 机构主页查询
- 发表记录比对

## 质量评估工具

### AMSTAR 2 (系统综述)

**领域**：
1. PICO 设计
2. 协议注册
3. 研究设计解释
4. 搜索策略
5. 研究选择
6. 数据提取
7. 偏倚风险评估
8. 合成方法
9. 结果解释
10. 利益冲突

### RoB 2 (随机试验)

**领域**：
1. 随机化过程
2. 偏离干预
3. 结局数据缺失
4. 结局测量
5. 选择性报告

### Newcastle-Ottawa (观察性研究)

**领域**：
1. 选择 (4 星)
2. 可比性 (2 星)
3. 结局 (3 星)

### GRADE (证据确定性)

**降级因素**：
1. 偏倚风险
2. 不一致性
3. 间接性
4. 不精确性
5. 发表偏倚

## 记录和报告

### 验证记录

**内容**：
- 来源信息
- 验证方法
- 验证结果
- 质量评估

**格式**：
```
来源：[作者, 年份, 标题]
DOI：[DOI]
验证方法：[方法]
验证结果：[结果]
质量评估：[评估]
```

### 质量报告

**内容**：
- 验证流程
- 质量分布
- 问题汇总
- 建议措施

**格式**：
```
验证来源数：[数量]
质量分布：[高/中/低]
主要问题：[问题列表]
建议措施：[建议]
```
